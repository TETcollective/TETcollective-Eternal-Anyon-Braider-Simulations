# TETcollective – Eternal Anyon Braider Simulations

Open-source Python code (QuTiP) for simulating primordial trefoil knot-induced anyonic braiding, entanglement entropy evolution, and topological catalysis for aneutronic fusion (20–40× enhancement prediction).  

This repo supports the preprint:  
**Eternal Anyon Braider Prototype: From QuTiP Simulations...** (Zenodo DOI: [inserisci])

## Contents (Python focus)

- `qu-tip_braiding_simulation.ipynb` → Main Jupyter notebook (Colab-ready) with full simulation pipeline
- `braiding_steps.py` → Standalone script for sequential braiding + plots
- `plots/` → Generated PDF outputs (entanglement entropy, <σz> expectation values)

## How to Run

1. Open in Google Colab: [link al tuo notebook se lo carichi]
2. Or locally: `pip install qutip matplotlib numpy`
3. Run the notebook/script → produces plots in `plots/`
